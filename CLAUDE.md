# TourOps Landing Page — Website Project

## What This Repo Is
This is the **touropsai.com marketing website**. It is deployed via Vercel.

## What This Repo Is NOT
This is NOT the TourOps operations/AI assistant build project. The `docs/` folder contains operational documentation that happens to live here, but **do not treat this repo as an ops project**.

## Rules
- When working in this repo, you are building a **website** (HTML, CSS, JS, images).
- Do NOT create or modify files related to: pipeline stages, conversation design, regression tests, GHL schemas, n8n specs, prompt compilers, or any other operational/AI build artifacts.
- Do NOT reference the docs/ folder contents as if they are part of the website project.
- The website domain is https://touropsai.com
- The brand name is **TourOps** (not "OurOps", not "Tour Ops").
